# Celebrity News Intelligence Dashboard

## 🚀 Features
- Crawl news from 50+ websites
- NLP-based celebrity extraction
- Relation extraction (celebrity + reason)
- FastAPI backend
- React frontend dashboard

## 🧱 Architecture
Crawler → NLP → FastAPI → React
